# Security Policy

## Signalement
Si vous découvrez une vulnérabilité (ex: lecture/écriture non désirée, path traversal, crash exploitable), merci d’ouvrir une issue **avec le tag `security`**.

## Périmètre
Ce projet est conçu en **lecture seule** sur `chat.log` et écrit des CSV dans `logs/`.
Il n’effectue **aucune injection/hook** dans Entropia Universe.
