# Configuration

Par défaut, le programme lit :
- `armes.ini`
- `markup.ini`

Les fichiers dans ce dossier sont des **exemples**.

Après clonage:
```bash
cp config/armes.ini.example armes.ini
cp config/markup.ini.example markup.ini
```

Puis édite `armes.ini` pour:
- `PLAYER.name`
- tes armes/amps
- MU ammo/weapon/amp si nécessaire
